## 📝 使用说明

### 查询知识点

```cypher
# 查看所有知识点
MATCH (kp:KnowledgePoint) RETURN kp.name, kp.difficulty LIMIT 10

# 查看特定分类的知识点
MATCH (ch:Chapter {name: "常识判断"})-[:BELONGS_TO_CHAPTER*]->(kp:KnowledgePoint)
RETURN kp.name, kp.content

# 查看题目及其解析
MATCH (q:Question) RETURN q.content, q.answer, q.analysis LIMIT 5
```
